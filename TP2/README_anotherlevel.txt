## Requisitos Previos
Asegúrese de tener instalado Python 3.6 o superior. Puede verificar su versión de Python ejecutando el siguiente comando en la terminal:
    python --version

## Librerías Requeridas
Las siguientes librerías son necesarias para ejecutar el código:
    - pandas: Manipulación y análisis de datos.
    - numpy: Operaciones matemáticas y manejo de arreglos.
    - matplotlib: Visualización de gráficos.
    - scikit-learn: Conjunto de herramientas para el aprendizaje automático que incluye:
        - train_test_split: División del conjunto de datos en entrenamiento y prueba.
        - KNeighborsClassifier: Clasificador basado en KNN.
        - DecisionTreeClassifier: Clasificador basado en árboles de decisión.
        - metrics: Evaluación del rendimiento del modelo.
        - KFold: Validación cruzada.
    - collections (opcional): Para trabajar con estructuras de datos como defaultdict.

Para instalar las librerías requeridas, ejecute el siguiente comando en la terminal:
    pip install pandas numpy matplotlib scikit-learn collections

## Ejecución del Código
El código debe ser ejecutado de forma secuencial sin saltear celdas, ya que los resultados de cada parte son necesarios para la siguiente. De lo contrario, se pueden obtener errores o resultados incorrectos.
